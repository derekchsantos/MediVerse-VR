# MediVerse: Sala de Emergência Virtual 🏥💙
Um ambiente de Realidade Virtual (VR) desenvolvido no Unity para treinamento médico e educação em saúde. Este projeto foi criado como parte da Residência em TIC 29 - Desenvolvimento de Metaverso.

## Sobre o Projeto
O MediVerse é uma experiência imersiva que simula uma sala de emergência hospitalar, onde o usuário pode explorar o ambiente, interagir com equipamentos médicos e aprender sobre protocolos de atendimento. O foco é demonstrar os fundamentos de XR, incluindo navegação, interação com objetos e feedback visual/sonoro.

## Requisitos
- **Unity:** Versão 6000.3.9f1 (LTS) ou superior.
- **SDK:** Meta XR All-in-One SDK.
- **Hardware:** Meta Quest 2/3 ou PC com Meta XR Simulator.

## Instalação e Configuração
1. Clone este repositório.
2. Abra o projeto no Unity Hub.
3. Instale o Meta XR SDK via Package Manager (`com.meta.xr.sdk.all-in-one`).
4. Configure o OpenXR em `Project Settings > XR Plugin Management`.

## Como Testar (Sem Headset)
Utilize o **Meta XR Simulator**:

1. No Unity, vá em `Meta > Simulator` e ative.
2. Pressione `Play`.
3. Use `WASD` para mover e `Mouse` para olhar.
4. Use `E` para interagir com objetos (Monitor Cardíaco).
